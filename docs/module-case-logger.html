<inner> Logger(options)

Interfaz con los métodos que se pueden usar para loguear.

Según el nivel de log que se quiera usar, hay distintos métodos disponibles. Todos ellos menos el nivel debug envían a Graylog.

Parameters:
Name Type Description
options String | Object

Opciones de inicialización del módulo. (Si sólo se pasa un string, será el namespace)

Properties

El identificador del script que llama al Logger.

(Opcional) Si enviar el mensaje a Graylog o no.
